**ACTIVE_SESSION_CONFIG** — exported const object:
```typescript
export const ACTIVE_SESSION_CONFIG = {
  ACTIVE_WINDOW_MS: 30 * 60 * 1000,   // 30 minutes
  IDLE_CUTOFF_MS:    5 * 60 * 1000,    //  5 minutes
  WORKING_SIGNAL_MS:      10 * 1000,   // 10 seconds — minimum safe value for WSL mtime precision
  TAIL_READ_BYTES:        16 * 1024,   // 16 KB
  CACHE_TTL_MS:            4 * 1000,   //  4 seconds (under 5s poll cadence)
} as const;
```

**tailReadJsonl(filePath: string, maxBytes?: number)** — reads last N bytes via byte-offset seek:
- `fs.statSync(filePath)` to get file size
- `readSize = Math.min(stat.size, maxBytes)` — default maxBytes is `ACTIVE_SESSION_CONFIG.TAIL_READ_BYTES`
- Allocate buffer, `fs.openSync(filePath, 'r')`, `fs.readSync(fd, buffer, 0, readSize, stat.size - readSize)`, `fs.closeSync(fd)`
- Split buffer to string, split on `\n`
- If `stat.size > maxBytes`, skip first line (partial from byte cut) — set `startIdx = 1`
- If `stat.size <= maxBytes`, start from line 0 — set `startIdx = 0`
- Filter empty lines from candidates
- Parse each candidate line with `JSON.parse` as `SessionMessage`
- If parse fails on the LAST non-empty line: set `hasIncompleteWrite = true`
- If parse fails on interior lines: skip (same as existing `forEachJsonlLine` behavior)
- Return `{ messages: SessionMessage[], hasIncompleteWrite: boolean }`

**inferSessionStatus(messages, fileMtimeMs, hasIncompleteWrite)** — pure function, no I/O:
- `ageMs = Date.now() - fileMtimeMs`
- If `ageMs > ACTIVE_SESSION_CONFIG.IDLE_CUTOFF_MS` → return `'idle'`
- If `ageMs <= ACTIVE_SESSION_CONFIG.WORKING_SIGNAL_MS` → return `'working'`
- If `hasIncompleteWrite` → return `'working'`
- Find last relevant message by walking messages array backward, skipping `type === 'system'` and `type === 'file-history-snapshot'`
- If no relevant message found → return `'idle'`
- If `last.compactMetadata` or `last.microcompactMetadata` → return `'working'`
- If `last.type === 'progress'` → return `'working'`
- If `last.type === 'user'` → return `'working'`
- If `last.type === 'assistant'`: check `message.content` — if `Array.isArray(content) && content.some(c => c && typeof c === 'object' && 'type' in c && c.type === 'tool_use')` → return `'working'`, else → return `'waiting'`
- Default fallback → return `'idle'`

**scanActiveFiles()** — filesystem scan:
- Import `getProjectsDir` from `./reader`
- `const projectsDir = getProjectsDir()`
- If `!fs.existsSync(projectsDir)` → return `[]`
- `const now = Date.now()`
- `const cutoff = now - ACTIVE_SESSION_CONFIG.ACTIVE_WINDOW_MS`
- Iterate `fs.readdirSync(projectsDir)` for project directories
- For each project dir, iterate `.jsonl` files
- `fs.statSync(filePath).mtimeMs` — if `> cutoff`, include in results
- Return array of `{ filePath, sessionId (filename without .jsonl), projectId (dir name), mtimeMs }`
